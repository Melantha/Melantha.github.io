1. absolute + 负margin

2. absolute + margin auto

通过设置各个方向距离都是0，然后再将margin设为auto

absolute + calc

4. absolute + transform 移动

使用css3新属性transform的translate, 其属性的百分比是相对于自身的宽和高进行移动。
